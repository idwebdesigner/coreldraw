Table tool adalah teknik hemat waktu yang kuat. Jika Anda hanya merancang brosur Anda dengan cara yang biasa, dengan menggunakan panduan, maka setiap kali Anda menambahkan atau mengubah elemen, Anda harus menyesuaikan keseluruhan desain. Dengan menggunakan table tool, Anda akan membuat pengeditan atau penggunaan ulang desain lebih cepat dan mudah.

      Desain Flyer Menggunakan Table Tool dan Live Text Preview


      Cara yang paling umum untuk membuat layout adalah dengan menggunakan panduan untuk menyelaraskan semua elemen dalam dokumen. Tapi ketika Anda perlu mengubah tata letak (misalnya, untuk menambahkan baris tambahan gambar), Anda harus melakukan tata letak sekali lagi.

      Anda akan menghemat banyak waktu dan usaha dengan menggunakan fitur table di CorelDRAW untuk menyusun dokumen Anda. Dalam tutorial ini, Anda akan menggunakan fitur table untuk membuat flyer.

      1. Mulailah dengan membuat sketsa flyer, dengan menggunakan pensil dan kertas, atau Smart drawing tool di CorelDRAW. Smart drawing tool secara otomatis mengubah garis kasar bentuk digambar tangan menjadi garis lurus atau kurva halus. Saat sketsa sudah siap, hitung jumlah kolom dan baris yang Anda butuhkan untuk tata letak.

      2. Buat tabel dengan pengaturan yang ditunjukkan di bawah ini dengan melakukan salah satu tindakan berikut:

      • Klik Table tool di toolbox, dan seret pada halaman untuk membuat tabel.
      • Klik Table > Create new table. di kotak dialog Create new table, ketik jumlah kolom dan baris, serta ukuran keseluruhan tabel.

       

      3. Klik Table tool di toolbox, dan ubah ukuran sel tabel sesuai kebutuhan dengan cara menyeret garis sel agar sesuai dengan tata letak. Anda dapat menarik garis saat kursor berubah menjadi panah ganda.

      4. Gabungkan sel bersama sesuai kebutuhan. Untuk melakukan itu, tarik melintasi sel untuk memilihnya (sel yang dipilih muncul dengan pola penetasan), lalu klik kanan dan pilih Merge cells di menu popup.

      Pada tabel layout di bawah ini, beberapa sel digabung untuk menampung gambar dan teks.

      Sekarang Anda siap mengisi sel dengan konten.

      5. Untuk menambahkan teks ke sel, klik Text tool di toolbox, klik di dalam sel, dan ketik. Untuk mengubah ukuran font atau font, pilih teks dengan Text tool, lalu gulir ke bawah kotak Daftar Font untuk menemukan font yang paling sesuai. Fitur Live Text Preview memungkinkan Anda melihat dulu perubahan teks yang Anda pilih dengan segera. Pada ilustrasi di bawah ini, ketika font Impact dipilih, teks yang dipilih diformat ulang dengan font baru.

      6. Untuk mengimpor gambar ke desain selebaran, klik Pick tool, Klik File > Import, cari file gambar di komputer Anda, dan klik Import. Klik halaman untuk menyisipkan gambar. Anda kemudian dapat memindahkan gambar ke sel tertentu dengan menyeret gambar dengan tombol mouse sebelah kanan. Saat sel target disorot, lepaskan tombol mouse dan pilih Place inside cell dari menu popup. Anda kemudian dapat menyesuaikan posisi dan ukuran gambar dengan menyeret gambar atau menyeret pegangan sudut.
      7. Akhirnya, hapus baris dari tabel. Untuk melakukannya, pilih tabel dengan Pick tool. Pada area Border di property bar, pilih All dari daftar kotak Pilihan Border, dan pilih None dari daftar kotak Outline width.
