      Bekerja dengan Lensa dan Efek di PHOTOPAINT


      Dalam tutorial ini, kita akan mengeksplorasi banyak manfaat menggunakan lenses dan masks untuk membuat penyesuaian gambar di CorelDRAW. Kita juga akan melihat cara membuat efek tonal khusus.

      Bekerja dengan Lensa

      Lensa, yang setara dengan lapisan penyesuaian di Photoshop®, memungkinkan Anda melihat efek khusus dan penyesuaian gambar tanpa mengubah piksel gambar secara permanen. Anda bisa memasang lensa ke seluruh gambar atau ke area yang dapat diedit dari mask. Saat menerapkan lensa ke area mask, Anda bisa menggunakan Object Pick tool untuk memposisikan kedua mask dan lensa di dalam gambar.

      Lensa juga menawarkan keuntungan untuk disesuaikan dengan cepat. Anda dapat melakukan penyesuaian terhadap lensa setiap saat.

      Karena lensa berada di atas gambar latar belakang pada lapisannya sendiri, Anda dapat dengan mudah melepaskan lapisan lensa yang tidak diinginkan dengan mengklik tombol Delete di sudut kanan bawah Object Manager docker.

      Catatan: Saat menyimpan gambar yang memiliki efek lensa yang diterapkan padanya, perhatikan bahwa banyak format file akan menggabungkan lapisan yang ada ke satu lapisan. Dalam kasus ini, efek lensa tidak dapat diedit lagi saat gambar dibuka kembali. Untuk mempertahankan kemampuan mengedit lensa, simpan file ini terlebih dahulu dalam format CPT, yang akan mempertahankan struktur lapisan. Anda kemudian dapat mengekspor gambar dalam format gambar yang dibutuhkan, seperti JPEG, PNG, BMP, dll.

       

      Menerapkan lensa pada gambar

      1. Klik File> Open, arahkan ke gambar Anda, dan klik Open.

      2. Klik Object> Create> New Lens.

      3. Di kotak dialog New Lens, pilih Scatter, dan klik OK. Lensa ini 'menyebarkan' piksel gambar, menciptakan tampilan yang dicat mirip dengan Pointillism.

      4. Di kotak dialog Scatter, lakukan percobaan dengan Vertical dan Horizontal sampai Anda mencapai efek yang diinginkan, dan klik OK.

      5. Untuk menyesuaikan kembali pengaturan lensa, klik dua kali ikon lensa di Object Manager docker, dan lakukan penyesuaian yang diperlukan pada kotak dialog Scatter.

      Menerapkan lensa ke area mask

      1. Buka gambar.

      2. Di toolbox, klik Rectangle Masktool.

      3. Di property bar, atur putaran kebiru-biruan mask dengan mengetikkan nilai di kotak Corner Radius.

      4. Tarik untuk menggambar mask persegi panjang pada area gambar.

      5. Untuk memberi mask tepi yang lebih lembut, klik Mask > Mask Outline > Feather. Di kotak dialog Feather, ketik nilai pada kotak Width, pilih Outside dari daftar kotak Direction, pilih Linear dari daftar kotak Edges, dan klik OK. Ringan tepi luar mask akan melunakkan transisi antara area mask dan gambar latar belakang.

      6. Klik Object > Create > New Lens. Di kotak dialog New Lens, perhatikan bahwa opsi Create lens dari mask sekarang diaktifkan secara otomatis.

      7. Pilih Gradient Map, dan klik OK. Gradient Map lens memungkinkan Anda untuk menerapkan warna pada gambar hitam-putih, atau untuk mengubah warna pada gambar berwarna.

      8. Di kotak dialog Gradient Map, klik dua kali tepi atas bilah warna gradien untuk menambahkan penanda warna baru, dan pilih warna baru dari Color picker.

      9. Lanjutkan menambahkan lebih banyak spidol warna sesuai kebutuhan, lalu klik OK.

      10. Untuk memposisikan kembali mask (dan lemsa), klik Object Pick tool lalu seret mask ke posisi baru.

      11. Untuk menyesuaikan kembali pengaturan lensa, klik dua kali ikon lensa di Object Manager docker.
