      Layers memberikan cara yang sangat efektif untuk mengatur objek yang Anda buat dengan CorelDRAW. Dengan menggunakan layers, Anda bisa mengurangi waktu kerja Anda, membuatnya lebih mudah untuk menangani pekerjaan yang ada, dan tingkatkan keakuratan desain Anda.

      Menggunakan Layers


      Object Manager docker (Gambar 2) menampilkan status struktur dokumen saat ini, sehingga memudahkan pengelolaan layers and objek. Ini menampilkan semua halaman dalam dokumen aktif dengan semua layers pada setiap halaman, dan semua objek ditempatkan pada setiap layer dari setiap halaman.

      Tombol di bagian atas docker membiarkan Anda melihat dan mengatur layers and objek.

      • Show Object Properties — memungkinkan Anda melihat properti dari setiap objek pada layer. Mengklik ikon + di samping nama halaman akan menampilkan semua detail objek.
      • Edit Across Layers — Memungkinkan Anda mengerjakan objek apa pun terlepas dari apakah objek berada pada layer aktif. Jika tombol dinonaktifkan, Anda harus membuat layer aktif agar bisa mengerjakan objek pada layer itu, sementara semua benda di layer lainnya akan terkunci untuk sementara.
      • Layer Manager View — Memungkinkan Anda mengelola hierarki layer dan mengubah urutan layer pada halaman aktif. Anda dapat menarik layer ke posisi yang diinginkan dalam hierarki.

      Tombol di bagian bawah docker membiarkan Anda membuat dan menghapus layer.

      • New Layer — Membuat layer baru pada halaman aktif.
      • New Master Layer (semua halaman) — menciptakan master layer baru di Master Page. Objek yang ditempatkan pada layer itu akan muncul di semua halaman dokumen.
      • New Master Layer (halaman ganjil) — menciptakan master layer baru di master page. Objek yang ditempatkan pada layer itu akan muncul pada semua halaman bernomor ganjil dari dokumen.
      • New Master Layer (halaman genap) — menciptakan layer master baru di Master Page. Objek yang ditempatkan pada lapisan itu akan muncul di semua halaman genap dokumen.
      • Delete Layer — Menghapus layer yang dipilih bersama dengan semua objek pada layer itu. Layer master default (Panduan, Desktop, dan Dokumen Grid) tidak dapat dihapus.

      Secara default, CorelDRAW memberi nama pada layer secara berurutan. Misalnya, Layer 2, Layer 3, dan seterusnya. Anda dapat mengganti nama layer untuk kenyamanan dan membuatnya lebih mudah dikenali saat Anda bekerja.

      Tombol properti layer di sebelah nama setiap layer membiarkan Anda mengelola properti layer itu.

      • Eye icon — Menunjukkan visibilitas layer. Mengaktifkan ikon mata membuat objek layer.
      • Printer icon — Matikan printability layer. Mengaktifkan ikon printer memungkinkan Anda mencetak dan mengekspor benda-benda di layer; Menonaktifkannya memblokir semua objek pada layer itu agar tidak dicetak dan diekspor.
      • Pencil icon — Matikan kemampuan edit dari layer. Mengaktifkan ikon pensil memungkinkan Anda memilih dan mengedit objek pada layer; Menonaktifkannya mengunci objek dan mencegahnya dipilih atau diedit.

      Tips: Biasanya, visibilitas dan kemampuan cetak harus diaktifkan atau dinonaktifkan bersamaan. Ingat, layer yang terlihat tidak dapat dicetak atau diekspor jika kemampuan cetak dinonaktifkan, dan layer yang tidak terlihat dapat dicetak dan diekspor jika daya cetak dapat diaktifkan. Layer yang terlihat namun tidak dapat dicetak dapat digunakan untuk menyimpan catatan dengan file. Saya menggunakan lyern seperti itu untuk menyimpan detail cetak dan informasi pekerjaan lainnya, sehingga saya bisa merujuknya kapan saja.

      Persiapan untuk memulai pekerjaan dengan lapisan layer

      Desain dokumen berlapis memerlukan beberapa pra-perencanaan sebelum Anda memulai dokumen CorelDRAW kosong. Saya lebih suka mencoret-coret desain pekerjaan dan penempatan benda pada karya seni.

      Membuat dokumen berlapis multi halaman

      Langkah pertama adalah mengatur ukuran halaman untuk pekerjaan dengan memulai dokumen baru (File > New). Secara default, CorelDRAW memulai sebuah dokumen baru dengan satu lapisan default yang diberi nama “Layer 1.” Setelah Anda memulai sebuah dokumen baru, pastikan bahwa Object Manager docker terbuka (jika tidak, klik Window > Dockers > Object Manager).

      Untuk pekerjaan kalender, saya telah menggunakan dua lapisan master dan empat lapisan lokal. Untuk membuat lapisan master, klik tombol New Master Layer semua halaman di bagian bawah bagan Object Manager docker. Objek yang ditempatkan pada layer master ini akan muncul pada semua halaman disain anda. Kemudian, buat jumlah yang dibutuhkan dari lapisan reguler atau lokal (dalam kasus ini, empat) dengan mengklik tombol New Layer. Anda sekarang memiliki satu halaman yang siap.
