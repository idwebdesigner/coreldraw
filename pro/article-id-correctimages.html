      Memperbaiki Gambar dengan Cepat dan Mudah di Corel PHOTO-PAINT®


      Sangat penting bagi seniman digital dan desainer grafis, mampu menyesuaikan dan memperbaiki gambar dengan cepat dan mudah. Pada artikel ini, kita akan melihat beberapa metode cepat untuk menuangkan gambar Anda di Corel PHOTO-PAINT® dalam beberapa langkah mudah.

      Menentukan Resolusi apa yang akan digunakan

      Tujuan gambar Anda menentukan resolusi untuk produk akhir. Anda dapat menggunakan tabel berikut untuk menentukan resolusi yang harus Anda gunakan:

      Output

      Resolusi*
      Web 72 dots per inch (dpi)
      Koran 125 dpi to 170 dpi
      Brosur dan majalah bermutu tinggi 200 dpi to 400 dpi
      Book layouts 175 dpi to 350 dpi


      Memindai Gambar di Corel PHOTO-PAINT

      Ada beberapa hal penting yang perlu dipertimbangkan saat memindai: kualitas pemindai cetak atau film, dan kualitas gambar yang sedang dipindai. Semakin tinggi kualitas pemindai dan gambar asli, semakin baik citra yang dikerjakan di Corel PHOTO-PAINT.

      Hasil asli berkualitas tinggi menghasilkan hasil terbaik, entah itu cetak atau film. Berikut adalah beberapa hal yang harus dicari:

      • Saturasi warna bagus
      • Kehalusan warna (tidak ada banding)
      • Definisi kontras yang halus dari sorotan ke bayangan, di mana sorotan menyimpan beberapa detail dan tidak meledak, dan bayangan mengandung detail dan tidak tiba-tiba jatuh ke hitam pekat.

      Sebaiknya pindai dokumen dengan resolusi lebih tinggi daripada yang dibutuhkan. Misalnya, jika memindai gambar untuk Web, pindai pada 150 dpi, bukan 72 dpi. Setelah gambar dipindai, kurangi ukuran gambar yang dibutuhkan.

       

      Memperbaiki Pola Moiré

      Saat memindai, terkadang akan menemukan pola moiré, garis-garis berwarna pelangi yang disebabkan oleh misalignment antara piksel dan perangkat pemindaian atau kamera. Pola moiré adalah hasil gambar asli yang memiliki area warna-warni yang ketat, warna-warni atau pola pengulangan yang ketat, seperti desain gigi anjing dalam kain.

      Dalam proses pencetakan majalah, titik-titik dilapisi untuk membuat gambar berwarna yang Anda lihat.

      Anda dapat mengurangi pola moiré dengan menempelkan gambar saat memindainya. Dengan gambar rata pada permukaan pemindai, putar di sudut dari vertikal, lalu pindai. Setelah pemindaian muncul di Corel PHOTO-PAINT, Anda dapat memutar gambar kembali ke vertikal dengan mengklik Image > Rotate dan memilih opsi rotasi.

       

      Skewing gambar seringkali bisa mengurangi pola moiré.

      Anda juga dapat dengan cepat dan mudah menghapus pola moiré di Corel PHOTO-PAINT dengan mengklik Effects > Noise > Remove Moiré.

      Di kotak dialog Remove Moire, anda dapat menyesuaikan moiré-removal settings.

      Memperbaiki Masalah Exposure dengan Cepat

      Corel PHOTO-PAINT memungkinkan Anda mengatasi masalah Exposure dengan cepat, yang disebabkan oleh terlalu banyak cahaya (overexposure) atau tidak cukup cahaya (underexposure) pada saat foto diambil. Berikut adalah beberapa teknik cepat yang bisa Anda gunakan.

      Menggunakan Image Adjustment Lab

      Image Adjustment Lab terdiri dari kontrol otomatis dan manual, yang disusun secara logis untuk koreksi gambar. Anda dapat mengakses Lab Penyesuaian Gambar dengan mengklik Adjust > Image Adjustment Lab. Untuk perbaikan cepat, klik Auto-Adjust (Atur Otomatis) di pojok kanan atas, lalu klik OK.

      Image Adjustment Lab

       

      Menyesuaikan Exposure di Objects Manage Docker

      Anda juga dapat menyesuaikan eksposur di Object Manager docker, yang dapat dibuka dengan mengklik Object > Object Manager. Di docker, seret objek latar belakang ke tombol New Object di bagian bawah docker untuk membuat salinan latar belakang.
      Kemudian, pilih Add atau Multiply dari Merge Mode di bagian atas docker. Mode Add meningkatkan exposure (seringkali cukup dramatis), sedangkan mode Multiply mengurangi exposure.

      kotak daftar Merge Mode (kiri) dan kotak Opacity (kanan) ada di bagian atas Objects Manager docker.

       

      Mode gabungan Tambah secara dramatis meningkatkan pemaparan.

       

      Mode penggabungan Multiply menurunkan eksposure.

      Anda dapat menggabungkan semua objek menjadi satu dengan mengklik Objects > Pilih Semua Objek dan kemudian klik Combine Selected Objects di bagian bawah Object Manager docker.

      Gambar asli exposure (kiri) dan gambar dengan mode tambah gabungan diterapkan (kanan)

      Memilih Format File Output

      Tujuan image-output yang umum adalah biro Web dan layanan. Format file untuk gambar akhir ditentukan oleh di mana gambar itu akan dilihat.

      Untuk Web, saya sarankan untuk menggunakan format JPEG untuk sebagian besar gambar. Untuk gambar 8-bit, sebaiknya gunakan format GIF atau PNG. Bergantung pada gambar, kompresi mungkin sedikit banyak atau kurang dalam file PNG daripada pada file GIF. Lihat artikel ini untuk mempelajari lebih lanjut tentang pengoptimalan gambar untuk web.

      Saat Anda bekerja dengan biro layanan, secara umum, TIFF adalah format file yang disukai. Anda juga bisa menggunakan Corel PHOTO-PAINT untuk membuat PDF, yang mendapatkan hasil yang sangat baik.

       

      Mengekspor ke PDF dari Corel PHOTO-PAINT dan CorelDRAW

      Ketika membuat sampul buku, gunakan Corel PHOTO-PAINT untuk melakukan koreksi warna pada foto-foto, yang kemudian di impor ke CorelDRAW®. Setelah menyelesaikan tata letak, simpan sebagai TIFF, yang menggabungkan foto ke latar belakang dan memberi warna hitam pekat. Dari situ, impor gambar itu ke Corel PHOTO-PAINT dan menggunakan perintah Publish to PDF untuk membuat gambar yang Anda lihat di bawah ini.

      Book finish cover layout

      Untuk mengekspor file ke PDF, klik File > Publish to PDF. Anda dapat mengakses berbagai pilihan dengan mengklik Settings pada kotak dialog Publish to PDF, yang membuka kotak dialog PDF Settings. Misalnya, Anda bisa menambahkan tanda tanaman atau registrasi, mengatur kompresi JPEG, dan menambahkan kata sandi ke file.

      Kotak dialog PDF Settings

      Untuk layout yang dibuat di CorelDRAW, Anda dapat mempermudah printer dengan mengkliknya File > Collect For Output pada CorelDRAW. Ini membuka wizard yang memungkinkan Anda mengumpulkan semua file dalam tata letak (seperti font), membuat PDF, dan menyimpan semua file dalam satu folder, sehingga memudahkan pengiriman file ke biro layanan. Dan kompres semua file bersama-sama sebelum mengirimnya.

      Dalam tutorial ini, Anda telah mempelajari beberapa teknik manipulasi desain, seperti cara menyiapkan layout, membuat pemindaian berkualitas tinggi, membuat koreksi pemaparan cepat, dan bekerja dengan biro layanan. Menerapkan pengetahuan ini akan mempercepat alur kerja dan membuat lebih mudah untuk mendapatkan hasil yang di cari.
