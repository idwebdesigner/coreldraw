      Tutorial ini menawarkan petunjuk sederhana untuk mengimpor file Adobe Illustrator (AI), PhotoShop (PSD) dan PDF ke CorelDRAW sehingga Anda dapat mengerjakannya. Bagus jika Anda berkolaborasi dengan perancang atau klien lain, atau jika Anda menggunakan produk Abode sebagai bagian dari alur kerja desain Anda sendiri.

      Impor Adobe Illustrator, PhotoShop dan File PDF


      Tips ini telah diperbarui untuk CorelDRAW® Graphics Suite X7. Sementara fitur serupa mungkin tersedia di rilis sebelumnya, tipsnya akan sepenuhnya berlaku untuk CorelDRAW Graphics Suite X7 dan versi yang lebih baru saja.

      CorelDRAW menyediakan dukungan yang disempurnakan untuk format file Adobe Illustrator (AI) terbaru dan termasuk file CS6, Portable Document Format (PDF), dan format file Adobe Photoshop (PSD) dan termasuk format file CS6. Sekarang lebih mudah untuk membuka, mengimpor, dan mengedit file dalam format ini. Beberapa perangkat tambahan penting meliputi:

      • Dukungan untuk format file PDF / A dan Acrobat 9 (PDF 1.7) & penomoran Bates
      • ukungan untuk menentukan ukuran halaman dengan objek selama ekspor PDF
      • CorelDRAW X7 menjaga tampilan lapisan penyesuaian dan efek palet Masker dari file PSD yang diimpor
      • Dukungan untuk papan berganda AI, pelestarian transparansi gradien, blob brush strokes dan bagian Preflight yang baru
      • Corel PHOTO-PAINT® X7 mempertahankan media vibrance, grayscale, dan filter filter yang dapat diedit dari file PSD yang diimpor dan diekspor.

      Anda dapat membuka file AI dan PDF dengan menggunakan perintah File > Open seperti file CorelDRAW lainnya, atau Anda dapat mengimpornya. Saat Anda membuka file AI dan PDF, file dibuka sebagai file CorelDRAW. Bila Anda mengimpor file AI dan PDF, mereka dibuka sebagai file CorelDRAW. Bila Anda mengimpor file AI dan PDF, mereka diimpor sebagai objek berkelompok dan dapat ditempatkan di manapun dalam gambar Anda saat ini.

      Untuk mengimpor file Adobe Illustrator (AI) atau Adobe Photoshop (PSD)

      Anda dapat mengimpor semua file AI & PSD dan termasuk file CS6. Untuk file yang tersimpan dalam Adobe Illustrator CS atau yang lebih tinggi, teks dapat diimpor sebagai teks atau kurva jika file tersebut kompatibel dengan PDF. Jika file tidak kompatibel dengan PDF, Anda harus terlebih dulu mengonversi semua teks menjadi garis besar dengan menggunakan perintah Type > Create Outlines di Adobe Illustrator agar teks dapat diimpor.

      1. Klik File > Import.
      2. Cari folder tempat file disimpan.
      3. Pilih AI – Adobe Illustrator dari kotak daftar Files of type.
      4. Klik nama file, dan klik Import.Jika Anda mengimpor file yang kompatibel dengan PDF yang berisi teks, kotak dialog Import PDF muncul. Di bagian Import text sebagai area, aktifkan opsi Text atau Curves. Jika Anda tidak yakin pilihan mana yang akan dipilih, lihat Untuk mengimpor file PDF. Ketika Anda mengimpor teks sebagai teks, Font substitution untuk kotak dialog font yang hilang akan muncul jika Anda tidak memiliki semua font yang digunakan dalam file yang terinstal. Pilih pengaturan yang Anda inginkan, dan klik OK.
      5. Saat kursor impor muncul, lakukan salah satu dari yang berikut:
        • Klik halaman gambar untuk mempertahankan file asli dan posisi pojok kiri atas tempat Anda mengklik.
        • Klik dan seret pada halaman gambar untuk mengubah ukuran file. Kursor impor menampilkan dimensi file ukuran saat Anda menyeretnya
          Menggambar halaman.
        • Tekan Enter untuk memusatkan file pada halaman gambar.


      Kursor impor menunjukkan nama file dan
      Dimensi file yang akan Anda impor.

      Grafik Adobe Illustrator diimpor ke dalam program sebagai kelompok objek. Klik Object > Group > Ungroup Objects (CTRL+U) untuk memanipulasi objek dalam grafik impor.

      Untuk mengimpor file PDF

      Anda bisa mendapatkan hasil terbaik untuk teks dalam dokumen PDF Anda dengan memilih mengimpor teks sebagai teks atau kurva. Petunjuk berikut akan membantu Anda membuat pilihan yang tepat:

      • Saat teks diimpor sebagai teks, font dan teks dipelihara, dan teks dapat diedit sepenuhnya sebagai teks artistik atau paragraf. Namun, beberapa efek dan format mungkin hilang. Pilihan ini disarankan jika Anda memiliki file PDF yang berisi blok teks besar, seperti buletin, dan Anda ingin memformat ulang teks atau menambahkan konten teks.
      • Bila teks diimpor sebagai kurva, tampilan teks, termasuk semua efek yang diterapkan padanya, dipelihara, dan setiap huruf diubah menjadi objek kurva. Dengan opsi ini, fitur pemformatan teks tidak dapat lagi digunakan untuk mengedit teks. Jika Anda memiliki file PDF yang berisi sejumlah teks kecil yang tidak memerlukan pengeditan, atau jika Anda tidak memiliki font yang digunakan dalam file PDF, Anda harus mengimpor teks sebagai kurva.
      1. Tekan Ctrl + I.
      2. Cari folder tempat file disimpan.
      3. Pilih PDF – Adobe Portable Document Format dari daftar kotak Files of type.
      4. Klik nama file, dan klik Import.Jika file ini dilindungi oleh kata sandi, ketik kata sandi yang valid di kotak Password.
      5. Di kotak dialog Import PDF, pilih salah satu opsi berikut di dalam Import text sebagai area:
        • Text – memungkinkan Anda mengedit dan memformat ulang teks dari file PDF. Jika salah satu font yang digunakan dalam file PDF hilang, pilih pengaturan yang Anda inginkan dalam Font substitution untuk kotak dialog yang hilang, dan klik OK.
        • Curves – Mengubah teks menjadi kurva, memungkinkan Anda mempertahankan tampilan teks asli

      Kotak dialog Impor PDF

      Jika Anda mengimpor dokumen multipage, pilih halaman yang ingin Anda impor, dan klik OK.

      Saat Anda mengimpor PDF multipage, kotak dialog Import PDF mengembang untuk membiarkan Anda memilih halaman yang akan diimpor.

      1. Saat kursor impor muncul, klik halaman gambar. Anda juga bisa mengklik dan menyeret untuk mengubah ukuran file, atau tekan Enter untuk menempatkan file di bagian tengah dokumen.
