      Manipulasi Gambar Menggunakan PHOTOPAINT


      Dalam tutorial ini, Anda akan belajar bagaimana menggunakan perangkat lunak PHOTOPAINT yang hebat untuk mengedit foto Anda.

      Saat Stefan Lindblad menuntun Anda meski proses manipulasi foto, Anda akan belajar tentang:

      • Menganalisis foto
      • Menggunakan Photo Filter Lenses
      • Menambahkan efek Halftone

      Sebagian besar foto yang kita ambil dengan kamera kita, bisa berubah menjadi sesuatu yang jauh lebih seru dan imajinatif. Aku ingat berjalan di jembatan milenium di London, saat seorang warga London lokal berjalan dengan kencan makan siangnya, berkomentar keras seolah dia ingin aku mendengarnya: “Saya tidak mengerti mengapa wisatawan mengambil semua foto ini setiap saat”. Nah London ini memang merindukan satu poin - mungkin saya adalah turis tapi saya selalu mengambil foto referensi juga. Ini adalah jenis DNA saya karena saya tidak pernah tahu apa pekerjaan desain & ilustrasi saya selanjutnya.

      Ambil foto ini misalnya, yang saya ikuti dalam perjalanan ke bekas distrik Berlin Mitte Timur di Jerman beberapa tahun yang lalu. Cukup biasa kan? Aku berjalan di jalanan dan sampai di lingkungan bohemian. Saya mengambil beberapa foto, dan yang satu inilah yang akan saya gunakan untuk tutorial ini, jadi ini sudah mulai digunakan.

      Menganalisis foto

      Melihat ke belakang di lingkungan ini di Berlin Mitte, saya pikir ini akan memberi contoh bagus untuk membuat sesuatu yang biasa sedikit lebih keluar dan imajinatif. Saya selalu memulai dengan melihat area yang kuat dan lemah dalam sebuah foto. Mereka bisa berubah menjadi aset besar saat pekerjaan berlangsung selama sebuah proyek.

      Corel PHOTO-PAINT alat ajaibnya

      Buka foto di Corel PHOTO-PAINT X7 dan lihat dari mana pencahayaan ini berasal. Pencahayaannya sangat hambar, hampir kusam. Tapi apa yang baik, adalah mungkin untuk bekerja dengan cahaya dan bayangan dan kontras. Saya melihat bayangan di bawah tempat duduk di luar ruangan dan pohon di sudut kanan bawah gambar dan akan ada dalam pikiran di kemudian hari.

      Saya mulai dengan membuat Objek Baru di Object Manager Docker (Object > Create > New Object), dan menggunakan Paint Tool (Toolbox > Paint tool), dan soft edge brush nib untuk menambahkan sinar matahari. Saya memilih Art brush > Large Soft dari daftar drop-down di Property Bar. Saya meningkatkan lebar nib sedikit menggunakan pengaturan Nib Size di Property Bar. Sekarang saya melukis 7 sinar matahari dari langit ke jalan menuju alun-alun, dengan warna putih. Kemudian saya menyesuaikan opacity dari brush stroke object dengan mengutak-atik Opacity slider di bagian atas Object Manager Docker.

      Selanjutnya, saya menggunakan Mask Tool untuk menutupi langit dan menyalin dan menempelkan ini (CTRL+C, CTRL+V), Untuk membuat objek grup dengan itu. Dengan cara ini saya hanya melakukan penyesuaian ke langit, dan bukan keseluruhan gambar, saat saya menggunakan fitur Lens Object Adjustment.

      Saya membuat 2-3 duplikat mask, sehingga saya bisa menerapkan perubahan yang berbeda ke langit, dan dengan cepat menghidupkan atau mematikan objek tertentu. Ketika saya selesai, saya hanya menghapus objek mask yang tidak ingin saya simpan. adi saya akan berakhir dengan satu objek mask langit. Sementara yang dipilih, saya menerapkan Tone Curve Lens (Object > Create > New Lens > Tone Curve) untuk meningkatkan kontras, membuat area gelap lebih gelap dan dengan S-Curve yang agak lembut (Nilai di jendela dialog: Hitam sekitar 51,54 dan Putih Sekitar 172, 162).

      Juga menambahkan efek Contrast Enhancement Lens Object, ditempatkan secara hierarkis di bawah objek Tone Curve dan dengan Input Value Clipping 12 – 255. Sekarang saya membersihkan bagian mask saya tidak ingin terus menggunakan Eraser Tool (Toolbox > Eraser Tool).

      Saya juga menerapkan efek Dodge & Burn, meningkatkan efek bayangan dengan Effects Tool > Dodge & Burn > Burn Highlights, di jendela dan fasad bangunan. Namun, sebelum saya menggunakan Dodge & Burn tool pertama saya menggunakan Mask tool ntuk menyalin dan menempelkan bagian gambar itu, dan menyimpannya sebagai objek terpisah di Object Manager Docker.

      Pada beberapa objek mask saya mengulangi efek Dodge & Burn seperti kuas cat. Ini memberi saya kebebasan untuk melukis bayangan dan kontras pada gambar dengan Effect tool, seperti melukis dengan cat warna air Gouache, Secara tradisional digunakan oleh ilustrator dan desainer sebelum usia grafis komputer. Setiap lapisan cat yang sama persis, atau dalam kasus ini, setiap lapisan Dodge & Burn, membuatnya lebih gelap. Dalam foto saya itu membuat plester dan semen terlihat lebih nyata dan menambah kedalaman.

      Di bangunan merah di sebelah kanan, saya mengulangi Dodge & Burn cukup ekstensif, menambahkan lebih banyak kontras, kedalaman dan bayangan yang lebih gelap. Hal ini membawa jendela lebih hidup.

      Melihat Parasol tepat di belakang sinar matahari, saya dapat melihat bahwa ini membutuhkan lebih banyak kontras. Saya menerapkan kontras menggunakan Effect Tool >Dodge & Burn > dan Dodge Highlights bukan Burn Highlights seperti yang kita gunakan sebelumnya. Ini meringankan bagian atas payung.

      Tenda luar, di atas meja di luar ruangan oleh rumah merah di sebelah kanan, kosong dan tanpa logo atau teks, yang memberi saya kesempatan bagus untuk menambahkan teks atau logo di atasnya. Aku pergi untuk “Kopi”. Dengan Text Tool (Toolbox > Text tool), dan warnanya putih, font San Serif yang cukup modern. Dengan Pick tool yang dipilih, saya menggunakan penyisipan distorsi untuk membentuk teks agar sesuai dengan tenda luar (klik pada ikon Distort di Property Bar). Agar terlihat lebih nyata, saya menerapkan opacity sementara saya menggunakan Eraser tool untuk menghapus apapun yang akan tumpang tindih dengan pohon, dan membuatnya sesuai dengan lebih baik. Setelah selesai, saya hanya mengeluarkan opacity dari teks. Saya menyelesaikannya dengan menyesuaikan opacity objek teks menjadi sekitar 30 menggunakan Opacity slider di Object Manager Docker.

      Menggunakan Photo Filter Lens

      Saya ingin menambahkan semacam nuansa kuning tua ke gambar, jadi di Object Manager Docker Saya memilih Lens Object > Photo Filter Lens (atau klik Object > Create > New Lens > Photo Filter). Kemudian saya lanjutkan dengan menambahkan Lens Object >Contrast Enhancement, dan juga objek Tone Curve Lens.

      Setelah menambahkan Photo Filter lens akhir, Saya melanjutkan. Semua bersama-sama, lensa ini menambahkan nuansa kuning tua yang lebih hangat dan lebih hangat ke keseluruhan gambar. Dan jika sekarang kita membandingkan citra kita saat ini dengan pencahayaan asli yang agak kusam, tiba-tiba kita merasa jauh lebih menarik.

      Menambahkan efek Halftone

      Untuk menambahkan efek Halftone saya membuat Gambar Baru sementara lainnya (File > New…). Setelah mengecat random brush strokes, saya menambahkan efek Halftone (Effects menu > Color Transform > Halftone). Artinya saya sekarang memiliki objek latar belakang putih dan yang lainnya dengan titik halftone. Melihat Color Transparency Tool di Toolbar, saya menyeretnya di atas warna latar putih, sehingga latar belakang transparan. Dari situ saya cukup copy paste objek ke gambar Berlin saya sebagai objek baru. Dengan cara ini saya bisa mengendalikan di mana dan bagaimana titik Halftone didistribusikan di atas gambar saya.

      Menambahkan seni & gambar impor ke foto

      Tapi belum selesai karena saya memiliki gagasan untuk menambahkan sesuatu ke fasad kosong yang menghadap ke alun-alun. Dan saya menemukan gambar vektor dari wajah anjing, seperti logo atau merek. Atau Street Art.

      Oleh karena itu saya mengimpor (Ctrl + I), gambar vektor anjing saya. Meskipun file CorelDRAW .cdr dalam format vektor, tidak masalah bagi Corel PHOTO-PAINT untuk membuka atau mengimpor.

      Klik (jangan seret ke tempat file), dan di Jendela dialog import pastikan Anda memilih opsi “Transparent background” sebelum mengklik OK. Saya mengimpornya sebagai objek bitmap tunggal. Saya mengurangi ukuran dengan menyeretnya ke ukuran menggunakan pegangan ukuran dan dengan Pick tool dipilih, saya meletakkannya di tempat di fasad rumah menggunakan ikonTransformation di Property Bar dan kemudian menyesuaikan pegangan yang sesuai. Penting untuk mendapatkan perspektif yang benar sehingga mengikuti perspektif fasad dinding. Dan kemudian menambahkan beberapa tekstur ke dalamnya juga. Dalam kasus ini saya menggunakan Textures di menu Effects bar. Untuk ini, saya menerapkan beberapa opacity untuk membuatnya lebih sesuai dengan dinding.

      Saya selesai dari gambar dengan menggambar dan melukiskan pennants dalam berbagai warna menggunakan Paint tool. Dan begitu selesai, foto yang dihasilkan tiba-tiba jauh lebih menawan daripada yang pertama saya ambil. Dari sesuatu yang cukup tanpa kegirangan hingga sesuatu yang jauh lebih menarik.
