      Dalam tutorial ini tertulis, Anda akan belajar membuat kartu Valentine di CorelDRAW. Ariel menuntun Anda melewati setiap langkah desain, menunjukkan beberapa trik rapi yang bisa Anda terapkan untuk membuat kartu romantis buatan sendiri Anda sendiri. Jatuh pingsan!

      Cara Membuat Kartu Hari Valentine


      Mari buat kartu Valentine's Day, dan tentu saja, dalam semangat perayaan, desain kita akan memiliki hati - murah hati.

      Menggambar hati dengan CorelDRAWÂ® sangat mudah. Saya hanya mengklik Basic Shapes tool dan, pada property bar, pilih bentuk jantung dari Perfect shapes picker.

      Saya dapat menekan Ctrl saat saya menyeret halaman untuk membuat hati yang proporsional, atau tahan Shift untuk menciptakan hati yang berpusat. Wajar saja, saya bisa membuat bentuk hati dengan menggunakan dua lingkaran dan satu segitiga, lalu mengelas tiga bentuk, tapi mengapa semua itu bekerja bila menggunakan bentuk yang sempurna begitu cepat dan mudah?

      Sekarang, saya memiliki bentuk dasar hati. Ada banyak cara saya bisa memodifikasinya; Sebagai contoh, saya dapat menambahkan perspektif dengan mengklik Effects > Add Perspective dan menyesuaikan grid perspektif. Saya juga bisa menggunakan Envelope tool untuk menciptakan efek yang serupa. Saat Anda membuat kartu Anda, gunakan imajinasi Anda. Ada cara tak terbatas untuk membuat desain unik dan orisinil dengan CorelDRAW. Tutorial ini tidak dimaksudkan untuk diulang secara step-by-step, namun berfungsi sebagai titik awal dan panduan.

      Untuk menambahkan bentuk hati yang lebih beragam, saya memilih salah satu dari hati dan menekan Ctrl + Q untuk mengubahnya menjadi kurva. Lalu, saya klik Window > Dockers > Fillet/Scallop/Chamfer dan oleskan fillet dengan nilai radius yang sesuai. Untuk contoh ini, saya sudah menerapkan fillet 5 mm dan 7 mm.

      Now, Sekarang, saya akan menggunakan Artistic Media brushes untuk menambahkan garis besar khusus. Saya bisa menyesuaikan pengaturan brush dari property bar, tapi saya suka menggunakan Artistic Media docker (Effects > Artistic Media) karena saya dapat menarik bentuk brush dari docker ke jendela gambar.

      Bereksperimenlah dengan brush dan setelan yang berbeda. Misalnya, Anda dapat mengubah lebar brush dengan menentukan nilai pada kotak Stroke width pada property bar. Anda bisa menggunakan bentuk yang sama dan menciptakan efek yang berbeda hanya dengan mengganti brush.

      Sekarang, saatnya menambahkan beberapa warna ke hati. Saya bisa menambahkan warna seperti halnya dengan objek apa pun, dengan mengklik sembarang warna untuk mengisi suatu objek, atau mengklik kanan semburat warna untuk menerapkan warna garis besar.

      Sekarang, saya menyebarkan hati ke halaman gambar.

      Pada titik ini, saya bisa mulai menambahkan beberapa kecerahan ke hati. Mesh Fill tool adalah favorit pribadi dan sangat berguna untuk tujuan ini. Ketika Anda mulai menggunakan Mesh Fill tool, itu bisa tampak menakutkan, jadi yang terbaik adalah memulai dengan objek sederhana (misalnya persegi panjang).

      Sedangkan untuk jantung ini, saya hanya perlu menambahkan simpul lalu mengatur warnanya. Saya bisa menambahkan warna dengan memilih node yang saya inginkan dan kemudian memilih warna swatch dari palet warna default. Jika saya menginginkan warna gelap dari warna tertentu, saya klik dan tahan tombol itu selama beberapa detik, sampai palet sekunder baru muncul. Aku bisa memilih warna yang berbeda dari palet sekunder itu.

      Setelah menambahkan sedikit kecerahan dan warna pada beberapa hati, saya menambahkannya ke halaman gambar kartu Valentine.

      Sekarang, saya bisa menggunakan Mesh Fill tool lagi untuk membuat latar belakang kartu. Di sini saya telah menggunakan nuansa warna yang sama, tapi itu bukan peraturan. Anda bisa menggunakan warna mana saja yang Anda inginkan, namun perlu diingat bahwa Anda menginginkan warna yang melengkapi gambar utama. Jika latar belakang terlalu gelap atau menggunakan terlalu banyak warna, maka teks atau citra mungkin kurang terlihat. Untuk contoh ini, saya telah membuat persegi panjang sekitar 106 mm dan tinggi 144 mm, dan kemudian menambahkan mesh fill sederhana dengan menekan tombol M dan memindahkan node tengah.

      Selanjutnya, saya mengatur pegangan simpul untuk membuat garis grid mesh yang lebih halus.

      aya menambahkan lebih banyak simpul dengan mengklik dua kali dengan Mesh Fill tool pada titik mana pun untuk membuat kolom dan baris khusus tambahan untuk grid mesh.

      Dengan semua penyesuaian yang dilakukan pada grid pengisi mesh, sekarang siap untuk beberapa warna. Untuk menambahkan warna, saya menggunakanShape tool (F10) untuk memilih beberapa node. Saya berkonsentrasi pertama di sekeliling mesh mengisi grid dan bekerja ke dalam. Untuk menghasilkan warna yang halus, saya tahan Ctrl dan klik swatch pada palet warna default. Dengan metode ini, saya menambahkan 10% nada warna yang dipilih setiap kali saya mengulangi aksinya. Saya menambahkan nuansa merah dan akhirnya menyentuh kuning ke kolom tengah.

      Pada titik ini, saya bisa menambahkan hati yang telah diciptakan sebelumnya ke latar belakang. Sebaiknya gunakan layer disini, yang akan mempermudah untuk kembali dan memodifikasi latar belakang jika perlu. Saya telah meletakkan hati yang penuh jala pada satu lapisan, hati Artistic Media di layer lainnya, dan latar belakang pada layer ketiga. Tentu saja, Anda dapat memilih untuk mengatur pekerjaan Anda namun Anda lebih suka.

      Aku masih butuh sedikit lagi hati untuk latar belakang. Alih-alih hanya menduplikat beberapa hati, saya bisa menggunakan sedikit trik untuk hanya menyalin bayangan beberapa hati. Bagaimana? Mudah. Pertama, saya menambahkan drop shadow ke bentuk hati yang sempurna dan mengubah warnanya menjadi merah. Kemudian, di bar properti, saya memindahkan slider Drop shadow opacity ke 100 dan memilih Outside dari kotak daftar Feathering direction.

      Sekarang, saya memecah drop shadow (Ctrl + K), jadi saya bisa menghapus bentuk aslinya. Saya hanya akan menggunakan drop shadow dengan transparansi. Saya memilih Multiply dari kotak daftar Merge mode di property

       

      Langkah selanjutnya adalah menggunakan Transparency tool untuk membuat jantung tembus dalam kotak kuning. Saya menduplikat jantung beberapa kali, mengubah urutan objek dan rotasi. Anda bisa menggunakan kreativitas Anda di sini untuk menghasilkan pilihan dan efek baru. Coba pilih salah satu hati tembus dengan Transparency tool dan pilih Divide dari kotak Merge mode pada panel property.

      Saya telah menambahkan beberapa hati tembus baru ke gambar utama.

      Sebelum menyelesaikan kartu, mari tambahkan beberapa teks dan buat beberapa modifikasi lagi dari bentuk dan warnanya. Pilihan Anda hampir tidak terbatas, sehingga Anda dapat dengan mudah menciptakan efek yang unik. Gunakan imajinasi Anda untuk memberi kartu itu gaya pribadi Anda sendiri.
